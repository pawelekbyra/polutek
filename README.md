# Projekt Dziennikarstwa Śledczego

Ten projekt, pierwotnie aplikacja typu TikTok, został przekształcony w jednostronicowy blog śledczy. Naszym celem jest opublikowanie przełomowego artykułu, który ma potencjał wywołania wielkiej afery medialnej.

Jesteśmy małym, niezależnym zespołem, a ta platforma stanie się jedynym źródłem ujawnienia tej historii. Chcemy, aby materiał był przygotowany profesjonalnie i stanowił wiarygodne źródło dla innych dziennikarzy.

## Cel

Ujawnienie prawdy w sprawie, która do tej pory była przedstawiana w mediach w sposób niepełny i jednostronny. Chcemy zapewnić platformę, na której dowody i fakty mówią same za siebie.
